# vue-tailwind-boilerplate

## Project setup

```
yarn install
```

### Boilerplate features

[x] vue 2
[x] tailwind
[x] vuex
[x] vue-router (history)

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
